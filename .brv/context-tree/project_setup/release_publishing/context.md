# Topic: release_publishing

## Overview
Covers how new Visualizador IA versions are published through GitHub releases and how the in-app updater detects and communicates available updates.

## Key Concepts
- Version bump in package.json
- GitHub release creation with gh CLI
- Manual installer generation
- Updater API polling and version comparison
- Testing update detection with older executables
